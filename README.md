# Numera

PWA latihan numerasi bertahap untuk anak usia dini.

App ini memakai pendekatan **Merdeka-aligned Singapore Math**:

- Kurikulum Merdeka sebagai peta kompetensi lokal.
- Singapore Math sebagai metode utama: Concrete, Pictorial, Abstract.
- Problem solving sebagai pengayaan menuju kesiapan olimpiade.

Soal dibuat oleh generator ringan, bukan bank soal besar, supaya variasinya banyak tetapi repo tetap kecil.

## Fitur awal

- Peta 12 level dari TK B sampai kelas 2.
- Generator soal penjumlahan, pengurangan, number bonds, nilai tempat, perkalian awal, pembagian awal, dan pola logika.
- Visual konkret sederhana untuk mendukung CPA.
- Progress lokal dengan bintang, jumlah soal, dan akurasi per level.
- Manifest dan service worker dasar untuk PWA offline-first.

## Jalankan lokal

```bash
npm install
npm run dev
```
